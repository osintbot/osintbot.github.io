# 👁️ Боты телеграм для пробива

### :man\_office\_worker: Физические лица, телефоны, почты

1. [LeakedInfo](https://vk.cc/cwaoTj) - бот для поиска информации о людях и компаниях. Доступ к государственным и слитым базам, а также к соцсетям и другим источникам. Предлагает поиск в 30+ странах.
2. [Users Box](https://vk.cc/cwapcp) - ищет данные из утечек. Умеет искать по ФИО, номеру телефона, email. Содержит базы всех известных утечек. Предоставляет 7 бесплатных дней использования.
3. [Leak OSINT](https://vk.cc/cwapgB) - универсальный бот с множеством функций. Ищет информацию из открытых источников: почту, телефоны, пароли, соцсети и многое другое. Предоставляет структурированные отчеты.
4. [Quick OSINT](https://vk.cc/cwapez) - мощный инструмент для поиска информации о людях и компаниях. Ищет почту, телефон, пароли, данные паспорта и СНИЛС, а также информацию из соцсетей и многих других мест. Предоставляет 2 бесплатных "пробива".
5. [Zernerda](https://vk.cc/cwapnF) - мощный бот с большой базой данных. Поиск по email, тел, авто, ФИО, IP и Twitter. Бесплатно можно использовать подписку в течение 1 дня.
6. [HimeraSearch](https://vk.cc/cwapq9) - ищет информацию о людях и компаниях из открытых и закрытых источников. Подробная информация о телефонах, email, авто, судебных делах и многое другое.
7. [Universal Search](https://t.me/UniversalSearchRobot) - поиск по телефону, почте, госномеру
8. [Nick Hunter](https://t.me/osint\_maigret\_bot) - поиск по никам
9. [Sova Search](https://t.me/SOVA\_SEARCHING\_BOT) - поиск аккаунтов по юзернейму, репутации почты и т.п.
10. [OSINT Handbook](https://t.me/QuickOSINT\_bot) - каталог OSINT ботов/сайтов и методов поиска
11. [Kazakhstan Check](https://t.me/ShtrafKZBot) - проверка ЮЛ и ФЛ по Казахстану
12. [Contact bot](https://ok.me/0FMG1) — указывает как именно записан номер телефона в контактной книжке
13. [Friend Graph](https://t.me/friendly\_graph\_bot) - по открытому профилю составляет дерево связей из друзей
14. [VK History](https://t.me/VKHistoryRobot) - показывает историю профиля с 2010 по 2021 года, фотки, что указывал на открытой странице&#x20;

16. &#x20;[Email Tracker](https://t.me/holehe\_s\_bot) - инструмент проверяет аккаунты каких сайтов зарегистрированы на искомый email адрес, поиск по 30 источникам.
