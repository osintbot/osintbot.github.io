# 🛳 OSINT — Поиск корабля

1. [MarineTraffic](https://www.marinetraffic.com/): Проект сообщества, предоставляющий данные о передвижении судов в реальном времени.
2. [Marine Vessel Traffic](https://www.marinevesseltraffic.com/): мониторинга морских судов, который использует картографические данные и обогащает их полезными фильтрами для удобного отслеживания различных типов судов.
3. [Global Fishing Watch](http://globalfishingwatch.org/map): дашборд для мониторинга человеческой деятельности в мировом океане: коммерческого рыболовства, перевалки грузов, случаев использования принудительного труда в море.
4. [VesselFinder](https://www.vesselfinder.com/): Сервис для отслеживания и поиска судов с актуальными данными.
5. [Shipping Explorer](https://www.shippingexplorer.net/ru): Сервис для отслеживания судов по всему миру, с приложением для Android.
6. [Ship Finder](https://shipfinder.co/): Сервис от Pinkfroot для отслеживания судов по всему миру.
7. [Shipspotting](https://www.shipspotting.com/): Сообщество, где любители делятся фотографиями кораблей.
8. [Cruise Mapper](https://www.cruisemapper.com/): Картограф для круизов.
9. [World Shipping Register](https://world-ships.com/): База данных всемирного регистра судоходства. Информацией о владельцах судов.
10. [Maritime Connector](https://maritime-connector.com/): Социальная сеть для моряков с вакансиями и списками работников.
11. [Superyachts](https://www.superyachts.com/): Поисковая база данных 100 крупнейших частных яхт.
12. [Moboxmarine](https://moboxmarine.com/): Информация о суперяхтах.
13. [Stolenboats](https://www.datatag.mobi/stolenmarine/Welcome.aspx): База данных украденных лодок и двигателей в Великобритании.
14. [gov.uk](https://www.gov.uk/): Судовой регистр Великобритании для торговых судов и бербоут-чартеров.
15. [ShipAIS](https://shipais.uk/): Информация о судах вокруг британского побережья.
16. [Prefixlist](https://www.prefixlist.com/): База данных цифровых кодов контейнеров.
17. [Track-Trace](https://www.track-trace.com/): Отслеживание контейнеров для 125 компаний.
18. [Maersk](https://www.maersk.com/): Сервис для отслеживания контейнеров.
19. [IMO](https://www.imo.org/): Общедоступная база данных грузов международной морской организации.
20. [Bic-code](https://www.bic-code.org/): Международные коды владельцев контейнеров (BIC Codes).
21. [Volza](https://www.volza.com/ports/): Международный код порта.
22. [ICC Commercial Crime Services (CCS)](https://www.icc-ccs.org/icc-commercial-crime-services-ccs): Все случаи пиратства и вооруженного ограбления с 2011 года.
23. [crewlist.org.uk](https://www.crewlist.org.uk/): База данных о кораблях и экипажах.
24. [Sanctionssearch](https://sanctionssearch.ofac.treas.gov/): Поиск в санкционном списке США.
25. [Equasis](https://www.equasis.org/EquasisWeb/public/HomePage?fs=HomePage): Информация о судах и компаниях, связанная с безопасностью.
26. [cpic-cipc.ca](https://www.cpic-cipc.ca/sbo-rba-eng.htm): Канадский сайт поиска лодок.
